## Description
This is a React-based personal event planner application built as a capstone project.  
The app allows users to register, log in, and manage personal or professional events. Users can add, view, edit, and delete events, all from a central dashboard. The application uses React Context API for state management and React Router for navigation.

## Features
- User registration with form validation
- User login with protected routes
- Add, view, edit, and delete events
- Dashboard displaying upcoming events
- Fixed navigation header
- Help section explaining how to use the app
- Responsive design using Bootstrap
- State management using React Context API

## Technologies Used
- React
- Vite
